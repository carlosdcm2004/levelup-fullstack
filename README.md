## Tecnologías
- Frontend: React + Vite
- Backend: Spring Boot + JPA + MySQL
- Base de datos: MySQL (XAMPP)
- Documentación API: Swagger
- Pruebas API: Postman

### Backend
```bash
cd backend/backendb_fullsatck-master
./mvnw spring-boot:run

cd frontend/react-main
npm install
npm run dev
```
